# IPL-Data-Analysis
This project analyzes IPL cricket player data using python, Pandas, and Matplotlib.

## Features
- Handles missing values in dataset
- Cleans IPL player data
- Creates data visualization charts

## Files Included  
- ipl_dataset.csv -> Original dataset with missing values
- cleaned_ipl_dataset.csv -> Cleaned dataset
- ipl_data_cleaning.py -> Python code
- ipl_visualization.py -> Python code for graph visualization
- runs_graph.png -> Bar chart of IPL player runs

## Technologies Used 
- Python
- Pandas
- Matplotlib

## Project Output
This project cleans IPL data and visualizes player performance using charts.
